# HTML5 Scripting

* 정식 명칭은 아님.
* DOM Scripting이 HTML5에 포함되었기 때문에 이를 구분하기 위함.
  * HTML5에는 마크업과 API만 있는 것이 아니고 DOM Script도 많은 부분 포함되어 있다.
